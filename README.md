## Начало работы

Достаточно склонировать проект из GitHub.

### Prerequisites

```
Git
браузер
```

### Установка и запуск

Пошаговый процесс установки и запуска

```
git clone https://github.com/Daria-chizh/neurostartup.git
открыть index.html
```
